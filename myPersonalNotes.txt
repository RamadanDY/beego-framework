Install Bee
Run:

>>>>go get -u github.com/beego/bee/v2@latest
For Go 1.18 and newer Go versions run:

>>>>>go install github.com/beego/bee/v2@latest


Check where Go installed bee

Run this command:

ls ~/go/bin

You should see something like:

bee

If you see bee, then it means Go installed it here:

~/go/bin/bee
2. Add Go bin to your PATH

Your terminal cannot find bee because ~/go/bin is not in your PATH.

Run this:

echo 'export PATH=$PATH:~/go/bin' >> ~/.bashrc

Then reload your terminal configuration:

source ~/.bashrc
3. Check again

Now run:

bee version

Create your first Beego project

Once bee works, you can create a project:

bee new myapp

Then enter the folder:

cd myapp

Run the application:

bee run

Your server will start on:

http://localhost:8080
